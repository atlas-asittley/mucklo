City Builder

Build Together. Trade Together. Thrive Together.

Game Overview

City Builder is a multiplayer tycoon game where players collaborate to build and manage cities. Work with others to gather resources, trade, and create a thriving metropolis together!

Combined Goods

Paper + Ink → Library
Iron + Wood → Toolmaker
Corn + Clay → Tavern
[Any Adv] + [Any Adv] → Marketplace

Trade Partners

  • River Traders - Balanced, wants Corn/Iron
  • Desert Caravan - Luxury lover, wants Stone/Clay
  • Mountain Folk - Metal focused, wants Timber/Iron
  • Coastal Merchants - Foodies, wants Processed Corn/Stone
